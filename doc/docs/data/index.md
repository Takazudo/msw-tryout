# Data Documentation

This section contains documentation about data structures, storage, and business logic for the MSW Tryout application.

## Overview

The application uses static gallery data with pagination support. Gallery items include image URLs, blurhash placeholders, and metadata for display.

## Data Categories

### Gallery Data

Documentation of gallery item structure, pagination mechanism, and data management for the image gallery.

### API Response Format

Details about API response structures, including pagination metadata and error handling formats.

## Storage Technology

Gallery data is served via **Netlify Functions**, providing:
- Serverless API endpoints
- Fast response times
- Simple JSON-based data storage
- Pagination support
